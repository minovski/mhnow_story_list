# README

## これはなんですか？

[Monster Hunter Now](https://monsterhunternow.com/ja)のストーリーとそのお題をリストアップしたもの。

## 使い方

[https://minovski.github.io/mhnow_story_list/](https://minovski.github.io/mhnow_story_list/)にテキトーなブラウザでアクセスする。

### 表示する情報

* チャプター
  * 区分（採取・討伐・緊急）
  * 具体的な内容
